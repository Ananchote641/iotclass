# Data Visualization.

Data Visualization ที่เราใช้คือ Grafana มาแสดงผล ที่มาจาก Sensor ต่างๆ โดยมี 1.อุณหภูมิ 2.ความชื่น 3.เวลา 4.แรงดัน 5.ควาามเข้มแสง และ มีการใช้ Plugin FlowCharting มาร่วมด้วยโดยการแสดงจะแสดงค่าาต่างๆ 
ที่สามารถแสดงเป็นทั้ง 

  Bar Chart: แสดงการเปรียบเทียบระหว่างหมวดหมู่ข้อมูล
  Line Graph: แสดงการเปลี่ยนแปลงของข้อมูลตามกาลเวลา
  Pie Chart: แสดงสัดส่วนของข้อมูล
  Heatmap: แสดงความหนาแน่นหรือระดับของข้อมูลในพื้นที่ต่าง ๆ
  Scatter Plot: แสดงความสัมพันธ์ระหว่างตัวแปรสองตัว
  
เป็นต้น การทำ Data Visualization ช่วยทำให้เราสามารถ วืเคราะห์ หรือ ป้องกัน หรือ มองภาพรวมที่ มีพื้นฐานมาจากขอมูลที่แสดงอยู่ได้ และที่สำคัญ สามารถติดตามสถานะได้

# Dashboard 1
ผ่านรวมของ Dashboard ที่สำคัญก็จะแสดง FlowCharting ที่เป็นแผนผังที่ แสดง sensor อุณหภูมิ 10ตัว อยู่ในแผนผัง และ มีการแสดง อุณหภูมิ 10 ตัวด้านนอก และ มีแรงดัน ที่มีแผนภูมิที่ใช้ต่าางกัน
-1.อุณหภูมิ ใช้เป็น Gauge Chart เพื่อสามารถแสดง ว่า อุณหภูมิ อยู่ในค่าที่เหมาะสมหรือไม่ และ ยังสามารถดูได้ว่า เกินค่าที่อันตรายหรือไม่ หรือ ร้อนเกินไปมั้ย สามารถตั้ง trashold ได้
-2.แรงดัน ใช้ bar chart เพื่อ สามารถดู ความเสถียน ของแรงดันว่าส่งมาคงที่หรือไม่
